# 先导
在项目开始前，我想根据我的开发经验，让浏览者明确自己的学习计划。
- **明确目标**
    1. 如果你需要一个交互式应用开发/图形引擎开发，甚至包括图形用户界面的开发经验，那么无论你是因为个人兴趣/参与面试，该项目都可以作为参考。
    2. 该项目主要复刻 Cherno 的教学：[游戏引擎系列](https://www.youtube.com/@TheCherno)。我个人采用合理合法的方式尝试复现其设计，面对问题时，使用当下合适的解决方式，并将开发时的思考与问题记录于 ***z笔记(notebook)*** 文件夹。
    3. 该项目的主要亮点在于：
        1. 将原有的代码以更整洁方式上传（分次）
        2. 笔记可供参考（我相信自己做的笔记还看得过去）
        3. 解决了由于时效性，有些代码现如今不可实现的问题。同时进行一些改进，修复潜在错误。
        4. 持续更新，虽然后续效率不高。我希望后续能新添一些功能，实现3D模拟（这是教程中没有的）。
- **你将会学到的知识**
    + 英语听力训练 
    + 开发者的常识、代码编写习惯、如何高效使用编辑器、甚至是快捷键 :-)
    + 中型项目的构架、项目维护、项目性能评估
    + C++特性、多数基础库的用途、C++模板等
    + 脚本使用
    + 一般 API 的使用流程
    + 图形渲染相关知识
    + UI 开发（ImGui）
    + 引擎的设计思路（层次、事件
    + 实体组件系统（entity component system）
    + 序列化/反序列化
    + 2D 物理碰撞模拟
    + C# 脚本导入
- **开发流程（时间）**
    基础的引擎开发（1~114），我一共花了9个月（除去假期以及休息），理论上至少每三天学习知识并进行一次提交。
- **知识基础**
    | 知识点  | 预计准备时间 | 推荐的视频/文档参考 |
    | ------ | ------ | ------ |
    | C++基础  | 2~3 个月 | [C++系列](https://youtube.com/playlist?list=PLlrATfBNZ98dudnM48yfGUldqGD0S4FFb&si=WQbZE6EAh3TQVbZI) |
    | OpenGL（或其他图形渲染API基础）  | 0.5~1 个月  |[OpenGL系列](https://youtube.com/playlist?list=PLlrATfBNZ98foTJPJ_Ev03o2oq3-GGOS2&si=4ikFgeFRrLHL4BYy) <br> [LearnOpenGL 文档](https://learnopengl-cn.github.io/) <br> 二者择一即可 |
    |线性代数 | 5 天| 此处涉及到线性代数的基本知识（矩阵） |

- **建议**
    + 建议学习周期：4~6个月。
    + 通过这个项目，我们可以训练出编程思维。而且在应对问题/Bug时，有先进的解决方式。这两点是从书本上学不来的，因为代码最高效的学习方式就是自己动手。
    + 该项目对图形学较为深入，内容难度不高。
      **所以在没有明确自己是否需要走游戏引擎相关岗业时，请认真思考是否需要把时间消耗于此。
      如果仅需要一个项目经验，可以选择学习其他C++项目，否则一定要把时间开销降在最低。（如同走马观花）**
